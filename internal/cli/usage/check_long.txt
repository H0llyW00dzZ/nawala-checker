Check whether one or more domains are blocked by Indonesian ISP DNS filters.

Domains can be provided as positional arguments, loaded from a text file
(--file), or both. Duplicates are removed automatically.

The text file format is one domain per line. Lines starting with '#' are
treated as comments, and blank lines are ignored.

By default, results are streamed to stdout as tab-separated text. Use
--json for NDJSON output (one JSON object per line), and --output (-o)
to write results to a file instead of stdout.

Examples:
  nawala check google.com reddit.com github.com
  nawala check --file domains.txt
  nawala check --file domains.txt --json
  nawala check --file domains.txt -o results.txt
  nawala check --config custom.yaml --file domains.txt --json -o out.json
